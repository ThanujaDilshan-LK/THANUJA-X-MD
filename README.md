## 🌟 About

**THANUJA-X-MD** is a powerful and fast WhatsApp Multi-Device bot built using **Node.js + Baileys**.  
It includes automation tools, group management, media downloading, and fun features.

---

## 🚀 Features

- ⚡ High Speed Performance  
- 📱 Multi-Device Support  
- 🎵 Media Downloader (YouTube, TikTok, Facebook)  
- 👥 Group Management (Kick, Add, Promote, Demote)  
- 🔐 Anti-Link & Anti-Spam  
- 🤖 Auto Reply & AI Features  
- 🎮 Fun Commands & Games  
- 🛠️ Easy Customization
